# highlight-reader

Project scaffold organized around the requested phases.

## Layout

- `capture/` handles clipboard, hotkey, and accessibility capture.
- `voice/` handles text-to-speech and playback controls.
- `agent/` handles interruptions and question/answer flow.
- `services/` coordinates capture, playback, and interrupts.
- `storage/` keeps history and database access.
- `ui/` contains tray and settings UI entry points.
- `utils/` contains shared logging and configuration helpers.

## Entry point

Run `main.py` from the repository root.
